# Guia de usuario de BetterMail AI

## Abrir el complemento

1. Crea un correo o abre una respuesta en Outlook.
2. Selecciona **Aplicaciones** en la barra del mensaje.
3. Abre **BetterMail AI**.

## Mejorar un borrador

1. Escribe el correo en Outlook.
2. Abre BetterMail AI o pulsa **Actualizar** si el panel ya estaba abierto.
3. Selecciona el tono: Profesional, Claro, Amable, Firme, Ejecutivo, Breve o Elegante.
4. Revisa el texto generado en **Mejorado**.
5. Usa una de estas acciones:
   - **Reemplazar**: sustituye el borrador actual y conserva la firma y la cadena anterior.
   - **Poner debajo**: agrega el resultado debajo del texto escrito.
   - **Copiar**: copia el resultado al portapapeles.
   - **Regenerar**: crea otra propuesta con el tono seleccionado.

## Generar una respuesta sugerida

1. Abre **Responder** en un correo recibido.
2. Deja el borrador vacio.
3. BetterMail AI detectara el contexto del mensaje anterior.
4. Revisa la respuesta sugerida y pulsa **Reemplazar** para colocarla en el editor.

BetterMail AI nunca envia el correo automaticamente. El usuario siempre debe revisar y pulsar **Enviar** en Outlook.

## Trial y BetterMail Pro

- El indicador **Uso** muestra las mejoras consumidas y disponibles.
- Cuando termina el trial, selecciona **Hazte Pro** para abrir la pagina de planes.
- BetterMail Pro incluye 300 mejoras por periodo mensual.
- Los pagos se procesan mediante PayPhone. BetterMail AI no almacena datos de tarjeta.

## Cancelacion

La cancelacion conserva el acceso Pro hasta terminar el periodo pagado. Mientras PayPhone habilita la administracion recurrente completa, solicita la cancelacion escribiendo a `gabriel.pacheco.developer@gmail.com` desde el correo asociado a BetterMail AI.

## Solucion de problemas

- **El panel no carga:** cierra BetterMail AI y vuelve a abrirlo desde Aplicaciones.
- **No detecta el texto:** pulsa **Actualizar** despues de escribir el borrador.
- **No puede mejorar:** confirma que existe texto o que estas respondiendo un correo con contexto.
- **El pago no aparece:** cierra la pagina de planes y abre nuevamente **Hazte Pro**.
- **El plan no se actualiza:** vuelve a Outlook, enfoca el panel y pulsa **Actualizar**.

No envies capturas con datos confidenciales, numeros de tarjeta ni el contenido completo de correos privados al solicitar soporte.
