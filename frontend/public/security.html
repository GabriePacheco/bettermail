Security / Seguridad

BetterMail AI is designed to process only the content needed to improve a draft or generate a suggested reply.

  • The user controls when text is applied.
  • The add-in does not send emails automatically.
  • Payments are processed by an external provider when enabled.
  • BetterMail AI does not store credit card numbers.
  • Data is transmitted over HTTPS.
  • Users may request data deletion through support.
