AppSource Test Notes / Notas de prueba

  1. Install BetterMail AI in Outlook on the web.
  2. Open a new compose window and launch the add-in.
  3. Write a draft and select a tone such as Professional or Firm.
  4. Confirm the generated text can be replaced, inserted below, copied and regenerated.
  5. Open an existing email reply with an empty draft to test suggested replies from context.
  6. Test aggressive non-physical wording such as "Respondeme ya o voy a escalar esto" and confirm BetterMail AI produces a firm professional version.
  7. Use trial until the limit is reached, then confirm the Pro call to action opens pricing.
  8. Use PayPhone sandbox data if provided by the publisher.

BetterMail AI does not send emails automatically. The user controls whether suggested text is applied.
